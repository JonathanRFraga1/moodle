Olá Mundo em PHP

Há duas formas de fazer o famoso "Hello World" em PHP
A forma convencional:

<?php

    echo "Hello World";

?>

E a "forma PHP":

<?php

    phpinfo();

?>

Olá Mundo em JavaScript

Agora veremos como executar o "Hello World" em js:

<script>

    alert('Hello World');

</script>

Olá Mundo em C++

Agora vamos ver como é feito um "Hello World" em C++:

#include <iostream>

int main(){

    std::cout << "Hello World!";

    return 0

}
